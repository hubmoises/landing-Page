# 🌐 Landing Page — Moisés da Silva Melo

Minha primeira Landing Page desenvolvida do zero, sendo o meu primeiro projeto publicado na Vercel. Criada com HTML, CSS e JavaScript, com integração de 3 APIs públicas do [BrasilAPI](https://brasilapi.com.br/).

##  Sobre o projeto

Página pessoal com seções de apresentação, habilidades e projetos práticos consumindo APIs reais. O projeto foi desenvolvido em um trabalho de n3 do professor "Willian Hubner" durante o 1º semestre do curso de Análise e Desenvolvimento de Sistemas (ADS) na Fasipe, Sinop-MT.

##  APIs utilizadas

| API | Descrição |
|-----|-----------|
| CEP | Busca endereço completo a partir de um CEP |
| Feriados | Lista feriados nacionais por ano |
| DDD | Retorna estado e cidades de um DDD |

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (Fetch API)
